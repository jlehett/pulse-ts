## Description

Add a "Solo" game mode where the player competes against AI opponents. The AI
should have distinct personalities that affect playstyle. Solo mode should be
available on both desktop and mobile. On mobile, the "Local" option should be
removed (it doesn't make sense on a single touchscreen) and replaced with Solo.

## Goal

Players can enjoy the game without needing another human player. Different AI
personalities provide variety and replayability — some aggressive, some
defensive, some erratic.
