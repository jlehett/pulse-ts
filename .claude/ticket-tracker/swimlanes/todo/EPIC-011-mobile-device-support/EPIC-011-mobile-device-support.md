## Description

Make the arena demo fully playable on mobile devices (Android and iOS browsers).
Currently the game is keyboard-only with hardcoded pixel-sized UI and desktop-optimized
rendering. This epic adds touch controls, responsive UI, and mobile performance tuning.

## Goal

A player on a mobile phone can join an online match, control their character via on-screen
touch controls, read all UI overlays comfortably, and maintain smooth frame rates.
